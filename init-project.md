# 提交仓库
## 首次提交
1. 本地
	```bash
	echo "# learning" >> README.md
	# 初始化目录为 git 项目
	git init

	# 加入缓存区
	git add README.md

	# 本地库提交
	git commit -m "init my learning"
	```
2. 远程
	```bash
	# 添加远程
	git remote add origin https://github.com/JoyceWang1/learning.git

	# 提交到远程
	git push -u origin master

	# 查看远程仓库
	git remote -v
	```
